# Live Audio Room

אפליקציית ווב פשוטה לשידור אודיו בזמן אמת בין משתמשים.

## תכונות
- יצירת חדר שידור חדש
- הצטרפות לחדר קיים באמצעות קוד
- שידור אודיו בזמן אמת
- שמירת חדרים בשרת

## התקנה והפעלה

### Frontend
האפליקציה רצה על GitHub Pages.

### Backend
1. התקן Node.js
2. התקן את התלויות:
```bash
cd backend
npm install
```
3. הפעל את השרת:
```bash
npm start
```

## טכנולוגיות
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- תקשורת: WebRTC
